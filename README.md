## Love Is Simple - CTF Web Challenge

**Author:** MD  
**Category:** Web  
**Difficulty:** Hard  
**Points:** 500

### Challenge Description

True love waits for the right moment. Some feelings can only be expressed at the perfect time.

Visit the application and confess your love... but only when the time is right.

### Flag Format

`TRACECTF{...}`

### Time Window

The challenge is only accessible during a specific time window each day. Find the right moment to confess your love.

### Deployment

This challenge is designed to be deployed on Render.com with automatic configuration via `render.yaml`.

### Files

- `app.py` - Flask application
- `requirements.txt` - Python dependencies
- `render.yaml` - Render deployment configuration
- `solution.py` - Solution script (for organizers)
- `WRITEUP.md` - Complete writeup (share after CTF)
